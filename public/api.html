API

This API shows all changes in location of the ship and the time when they occurred. It's meant to be a simple wrapper over the WorldState API that shows only the relevant information. Results are updated once a minute.
API is served from http://155.138.228.215:8080 only through HTTP.
There's only one endpoint: /api/get/:num.
The :num parameter can be left out, in which case it'll return just one result. The maximum number of results returned is 200.
Sample result:

    "result": "Gian Point"
  },
  {
    "time": 1576725431477,

  {

    "result": "Ganalen's Grave"

Sample call with 10 results
time is a UNIX timestamp.

Socket.io

You can listen to socket.io events on http://155.138.228.215:8080. The event identificator is simply event

Analyzer

Besides this tracker, a simple analyzer also exists. All it does right now is it calculates downtimes and uptimes based on up to 200 last results from the API, but I plan on expanding the functionality. Eventually.
Repo can be found here. It requires Dart 2.7.0 or above to run.
