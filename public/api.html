This API shows all changes in location of the ship and the time when they occurred. It's meant to be a simple wrapper over the WorldState API that shows only the relevant information. Results are updated once a minute.
API is served from https://anomaly-tracker.glitch.me
There's only one endpoint: /api/get/:num
The :num parameter can be left out, in which case it'll return just one result. The maximum number of results returned is 50.
